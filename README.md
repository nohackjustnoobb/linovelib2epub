# linovelib2epub

A simple script that fetches novels from the web and converts them into `epub` format.

## Quick Start

### 1. Clone the Repository

First, clone the repository to your local machine:

```bash
git clone https://github.com/nohackjustnoobb/linovelib2epub.git
```

### 2. Install Dependencies

Navigate to the project folder and install the necessary dependencies. Itâ€™s recommended to use a virtual environment for isolating the package installations:

```bash
cd linovelib2epub
pip install -r requirements.txt
```

### 3. Run the Script

Now you can run the script to start converting novels:

```bash
python cli.py
```
